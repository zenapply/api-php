## Installation

Install via [composer](https://getcomposer.org/) - In the terminal:
```bash
composer require zenapply/api-php
```

## Usage
```php
use Zenapply\Api\Zenapply;

$client = new Zenapply($API_TOKEN, "your_company_domain");

$positions = $client->positions->all();
```
